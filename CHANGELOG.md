# Changelog

## [0.1.0] - 2025-12-23

Initial release.

### Features

- Process spawning with `Px.spawn/3` and `Px.spawn!/3`
- Signal handling with `Px.signal/2` and `Px.signal!/2`
- Process waiting with `Px.wait/1` and `Px.wait/2` (with timeout)
- Pipe I/O with `Px.read/2`, `Px.write/2`, and `Px.close/2`
